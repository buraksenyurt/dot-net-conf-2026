Wireframe Tasarım Notları (Geliştirici Referansı)
  • AC-1: Varsayılan sıralama expiresAt ASC — en yakın bitiş tarihi üstte.
  • AC-3 / BR-2: isExpired=true olan veya ExpiresAt < UTC Now && status=Active olanlar Expired badge gösterir; DB güncellenmez.
  • BR-4: Sayfa boyutu seçici: 10 / 20 (varsayılan) / 50. pageSize query param olarak gönderilir.
  • BR-1: Tüm danışmanların opsiyonları listelenir (US-005 dashboard'undan farklı olarak kısıtlama yoktur).
  • AC-5: İlk yükleme ≤ 2 sn, filtre güncelleme ≤ 1 sn hedef performans.
  • API: GET /api/vehicle-options — VehicleOptionSummaryDto + PagedResult<T> response.
  • Sıralama: Sütun başlık butonlarına tıklanarak sortBy + sortDirection query param'ları değiştirilir.
  • Hata Durumu: API hatasında #error-banner görünür hale getirilir; tablo alanı boş bırakılmaz.
