# Frontend - Araç Envanter Yönetimi

Vue 3 + Nuxt 3 + TypeScript ile geliştirilmiş frontend uygulaması.

## Kurulum

```bash
# Bağımlılıkları yükle
npm install

# Development server başlat
npm run dev

# Production build
npm run build

# Production preview
npm run preview
```

## Teknoloji Stack

- **Nuxt 3** - Vue meta-framework
- **Vue 3** - Composition API
- **TypeScript** - Tip güvenli JavaScript
- **Tailwind CSS** - Utility-first CSS

## Proje Yapısı

```
src/frontend/
├── components/         # Reusable Vue components
│   ├── Header.vue     # Navigation header
│   └── Footer.vue     # Page footer
├── layouts/           # Layout templates
│   └── default.vue    # Default layout
├── pages/             # File-based routing
│   ├── index.vue      # Home page
│   └── vehicles/      # Vehicle pages
│       ├── index.vue  # List vehicles (TO BE GENERATED BY AI)
```

## Backend API Entegrasyonu

Backend API `http://localhost:5000/api` adresinde çalışacak:

- `GET /api/v1/vehicles` - Araç listesi
- `POST /api/v1/vehicles` - Yeni araç ekle
- `GET /api/v1/vehicles/{id}` - Araç detayı
